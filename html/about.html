TamperMonkey

Tampermonkey is s small tools that provides Greasemonkey script support for Google Chrome and Chromium Browser (version 5.0.0.0 or greater). It's API is fully compatible to GreaseMonkey, including GM_registerMenuCommand and GM_xmlhttpRequest with cross domain support. Furthermore there is a small GUI to manage all installed scripts. This includes editing, enabling, disabling, debug, uninstalling and so on). By default a small script is preinstalled that allows installing userscripts directly from userscripts.org.
