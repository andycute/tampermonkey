TamperMonkey

Tampermonkey is s small tools that provides Greasemonkey script support for Google Chrome and Chromium Browser (version 5.0.0.0 or greater). It's API is fully compatible to GreaseMonkey, including GM_registerMenuCommand, GM_getResourceText, GM_getResourceURL and GM_xmlhttpRequest with cross domain support. Furthermore there is a small GUI to manage all installed scripts. This includes editing, enabling, disabling, debug, uninstalling and so on). By default a small script is preinstalled that allows installing userscripts directly from userscripts.org.

API

TamperMonkey supports the following functions:

function TM_addStyle(css) aka GM_addStyle(css) add the given style to the document
function TM_deleteValue(name) aka GM_deleteValue(name) deletes 'name' from storage
function TM_listValues() aka GM_listValues() list all names of the storage
function TM_setValue(name, value) aka GM_setValue(name, value) set the value of 'name' to the storage
function TM_getValue(name, defaultValue) aka GM_getValue(...) get the value of 'name' from storage
function TM_log(message) aka GM_log(...) log a message to the console
function TM_getResourceText(name) aka GM_getResourceText(name) get the content of a predefined @resource tag at the script header
function TM_getResourceURL(name) aka GM_getResourceURL(name) get the base64 encoded URI of a predefined @resource tag at the script header
function TM_registerMenuCommand(name, fn) aka GM_registerMenuCommand(...) register a menu to be displayed at the TamperMonkey menu at pages where this script runs
function TM_openInTab(url) aka GM_openInTab(url) open a new tab with this url
function TM_xmlhttpRequest(details) aka GM_xmlhttpRequest(details) make an xmlhttpRequest, supports cross domain requests too
function TM_getTab(cb) get a object that is persistent as long as this tab is open
function TM_saveTab(tab) save the tab object to reopen it after a page unload
function TM_getTabs(cb) get all tab objects
function TM_getVersion() returns the TamperMonkey version string
function TM_installScript(script) install a userscript to TamperMonkey
function TM_run(fn, parentid, arg) runs 'fn(arg)' and catches errors to print a stack trace
this function is part of TamperMonkeys debug scripts option
function TM_addEventListener (event, fn) adds an event listener in the common way but wraps 'fn' with 'TM_run' to get stack traces
this function is part of TamperMonkeys debug scripts option
function TM_setTimeout (fn, time) sets an timeout in the common way but wraps 'fn' with 'TM_run' to get stack traces
this function is part of TamperMonkeys debug scripts option
